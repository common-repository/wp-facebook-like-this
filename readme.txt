== Description ==

This plugin allows your visitors to "like" your posts via new Facebook Like Function.

We also added a system that will automatically add the right og: tags in your blog header (regarding Facebook recommendations).

Available languages : english, french


== Installation ==

   1. Download the plugin or find it in Wordpress plugins repository from your admin page
   2. Upload it on your FTP (in /wp-content/plugins/ folder) or proceed to automatic installation
   3. Activate the plugin
   4. Go to Settings > WP-FacebookLikeThis
   5. Adjust settings
   6. Save settings
   7. Go back to your blog: you can now see a "I like" button at the end of your posts when they are fully displayed
